BANKING ANALYTICS PROJECT

Overview: 
This project analyzes customer financial data to understand
risk patterns, customer behavior, and revenue opportunities in banking.

Problem Statement: 
Banks must balance profitability with risk when
issuing loans. Poor decisions lead to defaults or lost revenue.

Dataset: 
3000 customers, 25 features including demographics, financial
data, and behavioral attributes.

Tools: 
Python (Pandas, NumPy), Matplotlib, Seaborn, MySQL, Power BI

Workflow: 
1. Data extraction from MySQL 
2. Data cleaning and validation
3. Feature engineering (Income bands, Engagement duration) 
4.Exploratory Data Analysis 5. Dashboard creation in Power BI

Key Insights: 
- Mid-income customers are primary revenue drivers 
- Financial data follows a Pareto distribution 
- Strong link between deposits and loans 
- Low customer loyalty indicates retention issues 
- Moderate credit usage suggests upselling opportunity

Conclusion: 
Data-driven insights can significantly improve lending
decisions, customer targeting, and profitability.

Future Scope: 
- Build ML model for loan prediction 
- Add real-time analytics 
- Perform time-series analysis
